# SaferHouse
A Smart Mobile Companion for Independent Elderly Safety and Community Support

## Overview
SaferHouse is an Android-based mobile application designed to support the safety of elderly individuals living independently. The system focuses on providing fast and reliable emergency response through simple and accessible features.

## Key Features
- Emergency alert via SMS and phone call
- Smart fall detection using smartphone sensors
- Audio-based distress detection (AI feature)
- Daily safety check-in monitoring
- Multi-contact alert with barangay escalation
- Works in low or no internet environments

## Purpose
The application aims to reduce delayed emergency response by allowing elderly users to quickly request help, while also enabling automatic detection of critical situations.

## Target Users
- Elderly individuals living alone
- Family members and caregivers
- Barangay/community responders

## Technology
- Android (Kotlin)
- SQLite (local database)
- Android APIs (SMS, Sensors, Microphone, Location)

## Note
SaferHouse is not a medical or diagnostic tool. It is designed only for emergency support and communication.
